# Télécharger et installer {#id1645E0O04MK}

À compter de la version 2024.2.0, Experience Manager Guides n’est disponible que sous la forme d’un module complémentaire automatisé pour Experience Manager as a Cloud Service.

Si vous utilisez la version de décembre 2023 ou une version antérieure, vous pouvez télécharger l’Adobe Experience Manager Guides à partir du référentiel GitHub et l’installer.
