# Package d’extension AEM Guides

Le package d’extension AEM Guides vous permet de personnaliser votre application en fonction de vos besoins. Vous souhaitez ajouter un autre bouton sur une barre d&#39;outils ? Un nouveau champ de saisie dans la zone de commentaires ? Une autre option dans le menu contextuel ? Maintenant, vous pouvez tout faire.

Accédez au [Guide d’installation](./integrating-customisations.md) pour comprendre comment utiliser ce package

> **_REMARQUE:_** ce package d’extension AEM Guides est pris en charge avec la version AEM Guides >= 4.3 (on-prem) et >= 2310 (cloud)
