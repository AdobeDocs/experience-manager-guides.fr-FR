# Configurer les modèles de rubrique et de mappage {#id181AH2003PF}

AEM Guides est fourni avec des modèles de plan Topic et DITA. Vous pouvez créer votre propre modèle personnalisé et le partager avec vos auteurs. Cette rubrique décrit le processus d’utilisation de rubriques et de modèles de mappage personnalisés.
