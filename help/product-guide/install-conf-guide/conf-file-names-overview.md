# Configurer les noms de fichier {#id214BD010GXA}

Cette rubrique contient des informations sur l&#39;utilisation des noms de fichier pour vos fichiers DITA. Dans le cas de systèmes basés sur l&#39;UUID, vous pouvez choisir de définir automatiquement un nom de fichier basé sur l&#39;UUID. Dans le cas de systèmes classiques, vous pouvez contrôler la liste des caractères valides autorisés dans les noms de fichier.
