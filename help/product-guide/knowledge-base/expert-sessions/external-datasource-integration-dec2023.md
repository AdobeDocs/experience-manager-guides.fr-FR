# Intégration d’une source de données externe pour AEM Guides : comment cela fonctionne-t-il ?

De nombreux clients d’entreprise disposent de données distribuées sur plusieurs bases de données, mais il est souvent nécessaire d’intégrer ces données dans les guides AEM en tant que source unique à partir de laquelle les données peuvent être traitées.
Adobe Experience Manager Guides comprend désormais des connecteurs prêts à l’emploi vers des sources de données externes, notamment JIRA, PIM, bases de données SQL et PLM, ce qui en fait un véritable Content Hub.

## Tons clairs

- **Date** - 14 Décembre 2023
- **Nombre de participants** - 80
- **Durée** - 60 minutes
- **Audience** - Clients, partenaires et employés d’Adobe

## Principaux points à retenir

Au cours de cette session, vous avez découvert les éléments suivants :
- Présentation des sources de données et de leur utilisation avec AEM Guides
- Configuration de la connexion pour diverses sources de données
   - Utilisation des outils Velocity dans les modèles de sources de données
   - Utilisation de modèles de mappage
- Afficher les données extraites du connecteur via l’éditeur web
- Utilisation du contenu extrait dans le contenu créé
   - Créer une rubrique à l’aide du générateur de rubriques
   - Insérer un fragment de contenu à partir de votre source de données
- Démonstration de travail de bout en bout de l&#39;intégration des guides AEM
   - mettre à jour les notes de mise à jour sur le site web et PDF à mesure que les modifications sont disponibles dans JIRA ;
