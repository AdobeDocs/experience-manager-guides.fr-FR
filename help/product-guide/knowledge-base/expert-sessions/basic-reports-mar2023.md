# Rapports dans AEM Guides

Dans une configuration d’organisation, vous souhaitez vérifier l’exhaustivité globale de votre contenu avant de pousser les documents pour qu’ils soient utilisés par les utilisateurs finaux. Un tel besoin devient encore plus essentiel dans les environnements multi-utilisateurs et à grande échelle où le contenu peut être réutilisé dans différents contextes. La validation de votre contenu ou de votre document est donc une nécessité. Dans le contexte du contenu structuré, certains types spécifiques de rapports sont généralement nécessaires.


## Session overview

Au cours de cette session, nous avons présenté les rapports prêts à l’emploi disponibles pour les utilisateurs. Nous avons largement couvert les sujets suivants :
- Rapports de document (carte) et de rubrique
- Rapports de métadonnées et options de gestion des métadonnées en bloc
- Rapports de réutilisation du contenu
- Exactitude de l’import de contenu
- Utilisation des fonctionnalités de reporting d’AEM Assets
