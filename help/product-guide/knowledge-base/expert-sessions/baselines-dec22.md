# Gérer le contenu à l’aide de lignes de base

La gestion des versions dans AEM Guides s’effectue à l’aide d’une ligne de base qui vous permet de choisir une version de vos rubriques et ressources, qui peut ensuite être utilisée pour la publication ou la traduction.

## Session overview

Au cours de cette session, vous apprendrez à utiliser des lignes de base pour gérer efficacement votre contenu.

## Enregistrement de session

>[!VIDEO](https://video.tv.adobe.com/v/3414172/version-management-release-management-baseline?quality=12&learn=on)

## Tons clairs

- **Date** - 8 Décembre 2022
- **Nombre de participants** - 64
- **Durée** - 60 minutes
- **Audience** - Clients, partenaires et employés d’Adobe

## Principaux points à retenir

Au cours de cette session, vous en apprendrez plus sur les sujets suivants :
- Différentes opérations de base dans AEM Guides
- Télécharger, publier et traduire à l’aide d’une ligne de base
- Bonnes pratiques relatives à l’utilisation d’une ligne de base
- Aperçu d’une nouvelle fonctionnalité à venir, **Créer des lignes de base dynamiques**
